# 页面 Loading

页面 loading 目前有三种，请根据具体项目和需求方要求使用相应loading，默认使用小黄鸡loading

# 使用

在 mobile-react-v4.1 的 src/components/route/route-loading/index.js 文件中通过修改 loadingImageType 改变 loading 类型
